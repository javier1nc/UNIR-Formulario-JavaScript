Realización de una página web utilizando Javascript

Realizar una página web utilizando JavaScript con un formulario que se debe validar.

El formulario contendrá campos de por lo menos 5 tipos diferentes.

Es obligatorio el uso de JavaScript para realizar comprobaciones que no sean posibles en HTML5.

Cuando se entregue el formulario se deberá generar un informe con las validaciones realizadas y mostrarlo por pantalla.

» Se podrá utilizar captura de eventos con JQuery o nativa Javascript (p.ej. onmouseover) a elección del alumno, pero no se podrán mezclar ambas tecnologías.

Comentario:

En el código JavaScript se definen las expresiones regulares que validan los campos del formulario mediante el control por IF que llama a una función que reporta los errores mediante un alert.
